图集错落九宫格实现方案

2018-03-27
  • css
  • 九宫格
  • 图片

最近在用vue仿小红书的h5版，在界面上唯一比较让人头大的就是这个伪九宫格(不知道有没有专业的称呼，我就这样叫了)的图集了。

难点分析如下：

  • 实现图片宽高一致
  • 尽量减少html标签
  • 尽量不使用js
  • 6n和12n情况下大图的左右位置不同
  • 最后一部分凑不齐9块的排布方案

小红书的效果如下：

小红书自己的方案是把每一个伪九宫格区域分为竖向左右两块，通过js判断数量动态决定添加的顺序为大小还是小大。
这样做的缺点有二：

  • 增加了很多无关内容的标签嵌套
  • 通过js进行计算导致逻辑繁琐

尝试过标签嵌套和绝对定位加js，最终通过vw和float实现了效果。

.imgs div {
  box-sizing: border-box;
  background-size: cover;
  width: 33.3vw;
  height: 33.3vw;
  padding: 1px;
  background-clip:content-box;
  background-position: center center;
  float: left;
}
.imgs div:nth-of-type(6n-5){
    width: 66.6vw;
    float: right;
    height: 66.6vw;
}
.imgs div:nth-of-type(12n-5){
    float: left;
}

    <div class="imgs">
        <div v-for="img in imgs" :style='`background-image: url(${img})`'></div>
    </div>
